`putout` plugin convert `spread` to `Array.from`.

## Install

```
npm i @putout/plugin-convert-array-copy-to-slice
```

## Rule

```json
{
    "rules": {
        "convert-array-copy-to-slice": true
    }
}
```

## ❌ Incorrect code example

```js
const places = [
    ...runPlugins(),
];
```

## ✅ Correct code Example

```js
const places = runPlugins().slice();
```
